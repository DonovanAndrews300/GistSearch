<!-- GETTING STARTED -->
### Instructions

1. Clone the repo
   ```sh
   git clone https://github.com/DonovanAndrews300/GistSearch.git
   ```
2. Install NPM packages
  ```sh
  cd /server npm install
  ```
  ```sh
  cd /client yarn install
  ```
3.Runing the app
```sh
  //server  npm start
  ```
  ```sh
  //client yarn start
  ```
